# Scripts

This folder is intentionally limited to data acquisition and data preparation code.

## data_downloading

Scripts that download or attempt to download external source data, including Fama-French factors, Welch-Goyal macro predictors, OSAP/GH-style characteristic proxies, and WRDS/Compustat inputs.

## data_processing

Scripts that convert raw files, merge CRSP chunks, construct CRSP-derived characteristics, rank the paper characteristics, and assemble model-input panels.
